# Employee Management System

## Project Description
A Spring Boot CRUD application to manage employee records.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Features
- Add Employee
- Get All Employees
- Get Employee By ID
- Update Employee
- Delete Employee

## API Endpoints

POST /api/employees

GET /api/employees

GET /api/employees/{id}

PUT /api/employees/{id}

DELETE /api/employees/{id}

## How to Run

1. Clone the repository
2. Configure MySQL
3. Update application.properties
4. Run EmployeeManagementSystemApplication.java
5. Test APIs using Postman
